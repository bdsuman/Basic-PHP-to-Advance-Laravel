## Operators and its types in php

### Arithmetic (গাণিতিক অপারেটর) 
>Arithmetic operators are used to perform arithmetic on numbers:

| Operator | Description                  |
    | -------- | ---------------------------- |
    | +        | Addition                     |
    | -        | Subtraction                  |
    | *        | Multiplication               |
    | **       | Exponentiation               |
    | /        | Division                     |
    | %        | Modulus (Division Remainder) |
    | ++       | Increment                    |
    | --       | Decrement                    |

### Assignment (নির্ধারণ অপারেটর)

- =, +=, -=, *=, /=

### Comparison (তুলনা)

- ==, !=, ===, !==, < (less), > (greater), <=, >=

### Increment / decrement

- !, &&, ||
